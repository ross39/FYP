What is Machine learning?

Machine learning is the study of computer algorithms that improve automatically through experience

How does it work?

Machine learning involves computers discovering how they can perform tasks without being explicitly programmed to do so. For simple tasks assigned to computers, it is possible to program algorithms telling the machine how to execute all steps required to solve the problem at hand; on the computer's part, no learning is needed. For more advanced tasks, it can be challenging for a human to manually create the needed algorithms. In practice, it can turn out to be more effective to help the machine develop its own algorithm, rather than have human programmers specify every needed step. We can do this by supplying the computer with lots of data. For example we can give thousands of imgaes of cats and dogs. Then over time Using these machine learning algorithms, the computer will learn to tell the difference between a cat and a dog. Then when you you give it an image of a cat or dog which it has never seen before, hopefully it will answer correctly

Why is this important?

The example I gave of telling the difference between cats and dogs is pretty trivial. However machine learning is far more than just telling the difference between cats and dogs. Machine learning is being used more and more in day to day life. Many people use machine learning applications without even realising it. These include virtual assistanst such as Siri and Alexa, traffic predictions on google maps, monitoring CCTV, social media services, email spam and malware filtering and online fraud detection. So you can see that machine learning isn't just something from a science fiction book. Now while modern machine learning is a long way away from science fiction sentient machines, It is revolutionising the modern world.

What are the main methods of machine learning?


Supervised learning

Lets say we have an x-ray image and we want to train a model to detect broken bones. We could gather a large dataset of x-ray images of broken bones and then label them as such. We could then train the machine to recognise the various patterns of broken bones in an x-ray. It is called supervised learning because the process of an algorithm learning from the training dataset can be thought of as a teacher supervising the learning process.

Reinforcement learning

Lets say we're playing a game of chess. Now in chess each player gets one move each turn so each player must consider their moves. There can be many good moves and many bad moves. However over time, by playing the game, we learn what constitutes a good move and what constititues a bad move. This is callsed reinforcement learning. Reinforcement learning is an area of machine learning concerned with how software agents ought to take actions in an environment in order to maximize the notion of cumulative reward.
